# 🎬 ЭМОДЖИКИНО

Угадай фильм или сериал по эмодзи.

## 🚀 Запуск

```bash
cd emoji-cinema
npm install
npm run dev
```

## 🎮 Как играть

1. Введи имя
2. Выбери режим: ВСЕ / ФИЛЬМЫ / СЕРИАЛЫ
3. Угадывай название по эмодзи
4. Используй подсказки (максимум 3)
5. За правильный ответ без подсказок — 5 баллов
6. Каждая подсказка снижает награду на 1 балл (минимум 2)
